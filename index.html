Iyengar Yoga Bangkok

Main page
Iyengar Yoga Bangkok was the first accredited Iyengar studio in Thailand. If you are a serious and experienced yogi this is a great studio for you, as there is a strong focus on precision, breathing and detail. Perhaps start at a ‘level 1’ class if you are unsure but looking for something a little more challenging.
